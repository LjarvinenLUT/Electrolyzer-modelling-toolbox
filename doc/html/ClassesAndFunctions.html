Classes and Functions

Classes

electrolyzerModel

Contains all the necessary information about the used electrolyzer model and enables its parametrization and model usage with simple-to-use methods.

viewWorkspace

Functions

Modelling functions

activation

Creates a func object for the activation overpotential.

concentration

Creates a func object for the concentration overpotential.

electrolyteParameters

Calculates saturated water vapor pressure and water activity for water solutions of KOH and NaOH at variable temperatures and molalities.

electrolyteWaterActivity

Provides the water activity for water solutions of KOH and NaOH at variable temperatures and molalities.

electrolyteWaterVaporPressure

Provides the saturated water vapor pressure for water solutions of KOH and NaOH at variable temperatures and molalities.

isCompleteStruct

Determines recursively if a given structure and all its substructures hav a values in all of their fields.

nernst

Creates a func object for calculating the open circuit potential in water electrolysis using Nernst equation. Combines the outputs from functions reversible and nernstPressureCorrection.

nernstPressureCorrection

Creates a func object for the pressure correction term of the Nernst equation.

nernstReversible

Creates a func object for the reversible potential term of water splitting reaction for the Nernst equation.

ohmic

Creates a func object for the ohmic overpotential.

waterVaporPressure

Calculates the saturated water vapor pressure in variable temperatures.

Utility functions

addValuesToStruct

Adds user specified values to the user specified fields in the given structure. The function doesn't add fields but only adds values to existing ones.

createSyntheticUI

Creates synthetic voltage--current pairs from an electrolyzer UI curve for testing purpose. A user-defined electrolyzerModel object can be used and standard deviation can be included in the created data.

fitUI

Parametrizes a given electrolyzer model using given UI curve data.

getConstants

Provides the accurate values of Faraday's constant, universal gas constant and number of electrons transferred in a single electrochemical water splitting reaction.

mergeStructs

Merges two structures recursively merging also any substructures.

mol2wtfrac

Convert concentration value in molality (moles of solute/mass of solvent in kg) to weight fraction (mass of solute/mass of solution).

pass

A do-nothing function for use as a placeholder when working with callbacks or function handles.

wtfrac2mol

Convert concentration value in weight fraction (mass of solute/mass of solution) to molality (moles of solute/mass of solvent in kg).

MCMC tool

Marchov Chain Monte Carlo (MCMC) tools provided by Marko Laine have been used for the determination of uncertainty in curve fitting when using the particleswarm optimization method. The licensing for the tools can be found under ../../electrolyzerModellingToolbox/Utils/mcmcstat/LICENSE.txt
