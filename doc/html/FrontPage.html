Electrolyzer Modelling Toolbox

Electrolyzer Modelling Toolbox is a tool aided to simplify the process of UI curve parametrization in water electrolysis applications. It provides means of defining the used model from an literature-based model library or by creating a custom modelling function. Different potential and overpotential terms are all included separately enabling creation of a combination that is the most relevant for each application. All the independently measured variables, variables that depend on the values of the independent ones, system related constants and modelling coefficients are stored with the model.
The toolbox includes easy-to-use methods for performing curve fitting to existing data to parametrize the model. The tool also provides an estimate on the uncertainty of the fit coefficient values. Usage of the model to calculate voltage based on given current values is simple with methods that utilise the variables that were earlier set for the model.
