A Word About Units

This toolbox does not recognise the units that are used for the current density and voltage inputs. Therefore, the user has to account for the used units when interpreting the UI curve coefficients provided by the toolbox. The writers of this toolbox encourage the user to use the following units:
  • Current density: (Total stack current divided for the effective electrode surface area in square centimeters)
  • Voltage: (Total stack voltage divided by the number of cells in series).
Using the recomeended units with the default models provided by the toolbox will provide fit coefficients with the following units:
  • Ohmic resistivity r: (Resistance of a single electrode divided by the electrode surface area)
  • Electron transfer coefficient Î±: (Unitless number representing a single cell)
  • Exchange current density : (Current divided for the effective electrode surface area)
  • Limiting current density : (Current divided for the effective electrode surface area)
Differing from these units will, naturally, result in changes in the output fit coefficients and may impair the fit performance as some of the modelling equations have been defined for a single cell.
